    Remarks

    Stĺpcom pridaným do zoznamu je automaticky nastavený index Index. Do zoznamu nie je možné pridať stĺpec, ktorý už patrí inému indexu. V takom prípade je vyvolaná výnimka System.InvalidOperationException.
