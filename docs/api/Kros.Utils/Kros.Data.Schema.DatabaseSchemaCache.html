    Class DatabaseSchemaCache

    Implementácia IDatabaseSchemaCache. Po pridaní loader-ov umožňuje načítavať schému databázy. Načítanie schémy je pomalá záležitosť a preto sa načítaná schéma kešuje. Pri ďalšej požiadavke na rovnakú schému sa už nenačíta z databázy, ale je vrátená z keše.

    Remarks

    DatabaseSchemaCache je potrebné inicializovať potrebným loaderom IDatabaseSchemaLoader, ktorý načíta schému. Zároveň každý loader musí mať špecifikovaný generátor kľúča ISchemaCacheKeyGenerator. Generátory kľúča pre rôzne typy databáz musia vytvárať navzájom rôzne kľúče, aby sa nestalo, že dva rôzne generátory vygenerujú rovnaký kľúč.

    Pre jednoduché použitie je implementovaná vlastnosť Default a nie je tak nutné si vytvárať vlastnú inštanciu keše.
