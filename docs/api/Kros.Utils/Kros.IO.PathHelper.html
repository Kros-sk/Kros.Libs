    Remarks

    Metóda pracuje podobne ako štandardná .NET metóda System.IO.Path.Combine(System.String[]), s pár upravenými detailami.

    Exceptions
    Type Condition
    System.ArgumentNullException

    Vstupný parameter parts má hodnotu null, alebo niektorá z jeho častí je null.
    System.ArgumentException

    Niektorý z reťazcov v zozname parts obsahuje nepovolené znaky, definované v System.IO.Path.GetInvalidPathChars.

    Vstupný reťazec, predstavujúci názov súboru.
