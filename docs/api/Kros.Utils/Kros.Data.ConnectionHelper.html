    Class ConnectionHelper

    Pomocná trieda pre otváranie spojenia na databázu. Trieda pri použití spojenie otvorí, ak je to potrebné a zase zatvorí, ak ho otvorila. V prípade, že spojenie už otvorené bolo, nerobí nič.
