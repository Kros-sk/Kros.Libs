Class DatabaseSchemaLoader

Pomocná trieda na jednoduché načítavanie schémy databáz. Obsahuje zoznam loader-ov pre konkrétne databázy a tak je možné použiť ju transparentne pre ľubovoľný typ databázy. Pri požiadavke je vždy načítaná aktuálna schéma, tzn. schémy sa nekešujú.

Remarks

Trieda obaľuje zoznam loader-ov rôznych typov databáz a umožňuje tak jednoduché načítanie schémy ľubovoľného známeho typu. Loadery je možné pridávať metódou AddSchemaLoader(IDatabaseSchemaLoader).

Trieda je určená na statické použitie pomocou vlatnosti Default. Štandardne dokáže načítať schémy SQL Server-a a MS Access.
