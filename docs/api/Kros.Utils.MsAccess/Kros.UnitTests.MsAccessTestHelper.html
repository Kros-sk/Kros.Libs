    Class MsAccessTestHelper

    Pomocná trieda pre unit testy, ak je potrebné mať pri testovaní reálnu databázu. Umožňuje jednoducho vytvoriť dočasnú databázu v ktorej sa bude testovať. Táto databáza je automaticky po skončení práce vymazaná.
