Remarks

Tabuľkám pridaným do zoznamu je automaticky nastavená databáza Database. Do zoznamu nie je možné pridať tabuľku, ktorá už patrí inej databáze. V takom prípade je vyvolaná výnimka System.InvalidOperationException.
