Remarks

Stĺpcom pridaným do zoznamu je automaticky nastavená tabuľka Table. Do zoznamu nie je možné pridať stĺpec, ktorý už patrí inej tabuľke. V takom prípade je vyvolaná výnimka System.InvalidOperationException.
